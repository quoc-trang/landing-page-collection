# 10 Days of UI - Landing Page Collection

A personal challenge to build 10 distinct landing pages using only HTML & CSS. This repository serves as a portfolio showcase of versatility in design, layout, and pure front-end fundamentals.

## The Challenge
The goal is to build 10 high-quality, responsive landing pages for different industries without relying on heavy frameworks.

7.  **Luxury Real Estate** - High-end imagery and elegance
8.  **Newsletter Signup** - High-conversion focus
9.  **Charity Campaign** - Storytelling and donation
10. **Retro Cyberpunk** - Experimental CSS art
